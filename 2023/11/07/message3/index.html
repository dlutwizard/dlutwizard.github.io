Zotero的安装

我们在进行科研或者做一个项目时常常需要阅读大量文献，有一个好用便携的文献管理工具相当重要。我目前经常使用的是Zotero,相比于其他文献阅读工具如endnote等，Zotero使用起来更加舒服，并且界面也很好看。接下来进入正题，Zotero的安装：

我们首先去到Zotero的官网下载地址 https://www.zotero.org/download/ 在这里选择你所需要的版本去下载。然后根据提示一步步安装。这里需要注意，如果你选择默认的安装方式standard方式去进行安装，那么它会自动安装到c盘且路径无法修改。安装好之后打开，如果你打开是默认英文，那么需要去修改一下

修改完成后这里就算是安装成功了，在这里我推荐去官网注册一个账号，在后面我们会用到。

如何使用

我们目前还无法直接从网上导入文献，需要在浏览器安装Zotero的插件

安装好之后启用插件，我们随便去一个学术网站试一下

这里我在谷歌学术，当上面的箭头指的地方变成黄色文件夹时，我们就可以点击扩展，然后会弹出图中的选项框，选择你想要导入的文献点击OK，此时文献就成功被我们导入到Zotero中了。

左边的侧边栏会显示文献的一些信息，展开对应导入的条目之后就是文献的pdf。我们双击打开pdf就可以进行阅读了。初期的一些配置就已经完成了，在后续的博客中我会写一些有关Zotero插件的推荐与使用，可以帮助我们更高效的阅读文献。

注意

如果我们在从浏览器导入时只有条目，却没有对应的pdf文章，那么我们需要在箭头的地方点进去修改

搜索findpdf

将括号中的内容修改为如下

{     "name":"sci-hub",     "method":"GET",     "url":"https://sci-hub.st/{doi}",     "mode":"html",     "selector":"#pdf",     "attribute":"src",     "automatic":true }

进行保存，然后我们在去浏览器试一下，此时就可以下载对应条目的pdf文件了。
